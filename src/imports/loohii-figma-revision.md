# 鹿绘AI (Loohii) UI 修改任务书（给 Figma 设计师）

> 基于当前已完成的第一版设计，以下是需要修改和补充的完整清单。

## 一、整体设计修改

### 1.1 配色调整

当前问题：品牌色单一，背景层级不分明。

**品牌色体系：**
- 主操作按钮：渐变 `#6366f1` → `#818cf8`（不要纯色平铺）
- 链接/可点击文字：`#a5b4fc`
- 选中态边框：`#6366f1`
- 状态色必须区分：
  - 成功/已完成：`#22c55e`
  - 进行中/生成中：`#f59e0b`（琥珀色，不要用品牌紫）
  - 失败/错误：`#ef4444`
  - 等待/未开始：`#71717a`

**背景层级（从深到浅）：**
```
最底层（画布/页面背景）: #09090b
侧边栏背景:              #0f0f11
内容区背景:              #111113
卡片/面板背景:           #18181b
输入框/悬浮面板:         #1f1f23
Hover 状态:             #27272a
```
每层之间必须有可感知的明度差。

**边框：**
- 弱分割线：`#1f1f23`
- 卡片边框：`#27272a`
- 选中/聚焦边框：`#6366f1`

### 1.2 字体规范调整

**基准字号从 16px 降到 14px。** 这是专业创作工具，需要更高信息密度。

| 级别 | 大小 | 字重 | 用途 |
|------|------|------|------|
| 页面标题 | 24px | Semibold | "我的项目"、"生成记录" |
| 区块标题 | 16px | Medium | "充值积分包"、"消费记录" |
| 卡片标题 | 14px | Medium | 项目名、节点标题 |
| 正文 | 14px | Regular | 描述、对话内容 |
| 辅助文字 | 12px | Regular | 时间、标签、状态 |
| 微型文字 | 11px | Regular | 角标、积分数 |

### 1.3 圆角统一

| 元素 | 圆角 |
|------|------|
| 按钮 | 6px |
| 输入框 | 8px |
| 卡片 | 12px |
| 弹窗/面板 | 16px |
| 头像 | 圆形 |
| 标签/Badge | 9999px |

### 1.4 间距

基准单位 4px。组件内间距 8-12px，组件间间距 16-24px，区块间间距 32-48px。

## 二、侧边栏修改

**当前问题：** 图标太小，选中态不明显，缺少层级感。

**修改：**
- 图标尺寸：20px
- 点击区域：40×40px（居中）
- 选中态：左侧 3px 品牌色竖条 + 图标颜色变为 `#fafafa` + 背景 `#1f1f23`
- 未选中：图标颜色 `#71717a`
- Hover：图标颜色 `#a1a1aa` + 背景 `#18181b`
- 项目内子导航（设定/画布/记录）用 2px 左缩进 + 更小图标(16px)表示层级
- 主导航和底部设置之间加 1px 分割线
- 侧边栏宽度：收起 56px，展开 200px

---

## 三、顶栏修改

**修改：**
- 高度：48px
- 积分显示：去掉 emoji，改为自定义图标 + 数字 + "积分"文字，包裹在 `#1f1f23` 圆角胶囊(radius 9999px)内
- 通知铃铛：右上角加红色圆点（有未读时显示）
- 头像：32px 圆形，点击弹出下拉菜单
- 面包屑：`鹿绘AI (Loohii) > 项目名` 用 `>` 分隔，项目名可点击

## 四、逐页修改

### 4.1 Dashboard（项目列表）

**修改：**
- Tab 栏（全部/最近/收藏）改为**胶囊式**（选中项有背景色 `#1f1f23` + 圆角），不要下划线式
- 项目卡片：
  - 封面占 60%，信息区占 40%
  - 信息区增加：分镜完成进度（如"3/8 分镜"）
  - Hover 时封面上浮现操作层：进入 / 重命名 / 复制 / 删除（4个图标按钮）
  - 卡片底部信息：比例 Badge + 风格 Badge + 更新时间
- "新建项目"卡片：高度与项目卡片一致
- 增加**空状态设计**：0 个项目时显示插画 + "创建你的第一个项目" + CTA 按钮

---

### 4.2 全局设定（Setup）— 通用修改

**修改：**
- 步骤条圆圈缩小到 28px，连接线用 1px 细线
- 所有步骤内容区最大宽度 **640px 居中**
- 底部"上一步/下一步"按钮 **sticky 固定在底部**，不随内容滚动
- 步骤可跳转（点击步骤条任意已完成/当前步骤）

#### Step 1: 基本信息
- 增加"项目封面"上传区（可选），避免页面太空
- 上传区：虚线框 + 拖拽提示 + 点击上传

#### Step 2: 画面比例
- 卡片尺寸加大到 **180×200px**
- 图标区域改为**实际比例矩形示意**（用半透明品牌色填充的矩形），不要小图标
- 选中态：品牌色边框 + 背景微亮 `#6366f1/10%` + 右上角圆形勾选✓ + 微放大(scale 1.02)

#### Step 3: 风格预设
- 网格改为 **4-5 列**（当前 4 列 OK，但卡片要更紧凑）
- 预设数量增加到 **至少 8-12 个**（当前只有 4 个太少）
- 补充预设：写实电影、日系动画、欧美卡通、像素风、油画风、扁平插画、3D 渲染、水彩风
- 每个卡片必须有**真实预览图**（不能显示 alt 文字）
- 最后一个位置放"+ 自定义"卡片（虚线框 + 加号）
- 选中卡片：品牌色边框 + 右上角圆形勾选✓

#### Step 4: 导演指导
- 两个 textarea 高度增加（至少 5 行可见）
- 增加 placeholder 示例文字（中文）
- 底部提示文字保持

### 4.3 节点画布（Canvas）

**修改：**

#### 工具栏
- "添加节点"改为**下拉按钮**，展开显示 4 种节点类型：
  - 🎬 分镜节点
  - 👤 角色节点
  - 🖼️ 资产节点
  - 📤 输出节点
- 每种类型带图标 + 名称 + 简短描述

#### 连线样式
- 角色→分镜：虚线 + 品牌色 `#6366f1`（表示"参与"）
- 分镜→分镜：实线 + 灰色 `#3f3f46`（表示"顺序"）
- 任意→输出：粗实线(2px) + 渐变色（表示"汇聚"）

#### 节点设计
- 分镜节点：当前设计 OK，保持
- 角色节点：加宽到 220px，头像改为 48px，增加底部输出连接点
- **新增：资产节点** — 缩略图 + 文件名 + 类型标签(图片/视频/音频)
- **新增：输出节点** — 显示格式/时长/完成进度 + "合成"按钮

#### 节点选中态
- 品牌色边框 + 外发光效果（`box-shadow: 0 0 0 2px #6366f1, 0 0 12px #6366f1/30%`）

#### Minimap
- 移到**左下角**（避免和 Agent 浮动按钮冲突）

#### 新增：节点右键菜单
- 生成 / 重新生成 / 编辑 / 复制 / 删除
- 带图标 + 快捷键提示
