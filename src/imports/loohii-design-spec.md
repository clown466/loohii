# 鹿绘AI (Loohii) 产品设计规范文档

> 本文档供 Figma 设计师使用，包含完整的页面结构、交互逻辑和设计规范。

---

## 一、产品概述

鹿绘AI (Loohii) 是一个 AI 驱动的动画/漫画创作 SaaS 平台。用户通过节点画布组织创作流程，AI Agent 辅助生成图片、视频、音频等内容。

**核心用户：** 独立创作者、小型动画工作室、漫画作者
**平台：** Web 端优先（桌面为主，平板适配）
**语言：** 中文为主，后续支持英文

## 二、设计系统规范

### 2.1 设计方向

**关键词：** 专业、沉浸、创意工具感
**参考产品：** Figma（面板密度）、Runway（AI 创作工具）、ComfyUI（节点画布）、Linear（简洁 SaaS）

### 2.2 配色

#### 暗色模式（主模式）

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 L1 | `#09090b` | 最底层背景 |
| 背景 L2 | `#141416` | 卡片/面板背景 |
| 背景 L3 | `#1c1c1f` | 输入框/悬浮面板 |
| 背景 L4 | `#252528` | Hover 状态 |
| 边框弱 | `#27272a` | 分割线 |
| 边框强 | `#3f3f46` | 卡片边框 |
| 文字主 | `#fafafa` | 标题/正文 |
| 文字次 | `#a1a1aa` | 描述/标签 |
| 文字弱 | `#71717a` | 占位符/禁用 |
| 品牌色 | `#6366f1` | 按钮/选中/链接 (Indigo) |
| 品牌浅 | `#818cf8` | Hover |
| 成功 | `#22c55e` | 完成状态 |
| 警告 | `#eab308` | 进行中 |
| 错误 | `#ef4444` | 失败/删除 |

#### 亮色模式（备选）
设计师可基于暗色模式反转，优先级低，后续再做。

### 2.3 字体

| 用途 | 字体 | 备选 |
|------|------|------|
| 正文 | Inter | system-ui |
| 中文 | 思源黑体 (Noto Sans SC) | PingFang SC |
| 代码/数字 | JetBrains Mono | Fira Code |

### 2.4 字号

| 级别 | 大小 | 行高 | 用途 |
|------|------|------|------|
| Display | 36px | 40px | 落地页大标题 |
| H1 | 24px | 32px | 页面标题 |
| H2 | 20px | 28px | 区块标题 |
| H3 | 16px | 24px | 卡片标题 |
| Body | 14px | 20px | 正文（基准） |
| Small | 12px | 16px | 标签/辅助文字 |
| Tiny | 11px | 14px | 角标/时间戳 |

### 2.5 间距系统

基准单位 4px，常用：4 / 8 / 12 / 16 / 20 / 24 / 32 / 40 / 48 / 64

### 2.6 圆角

| 元素 | 圆角 |
|------|------|
| 按钮 | 6px |
| 输入框 | 8px |
| 卡片 | 12px |
| 弹窗/面板 | 16px |
| 头像 | 圆形 |
| 标签/Badge | 9999px (全圆) |

### 2.7 阴影

暗色模式下阴影不明显，主要靠边框和背景层级区分。
- 浮动面板: `0 8px 32px rgba(0,0,0,0.5)`
- 下拉菜单: `0 4px 16px rgba(0,0,0,0.4)`

### 2.8 动效

- 过渡时长: 150ms (微交互) / 300ms (面板展开) / 500ms (页面切换)
- 缓动: `ease-out` 为主
- 画布缩放/平移: 60fps 流畅

## 四、通用布局

### 4.1 App Shell (登录后所有页面)

```
┌──────────────────────────────────────────────────────┐
│ Logo │ 项目名/面包屑          │ 积分余额 │ 通知 │ 头像 │  48px
├──────┼───────────────────────────────────┼──────────┤
│      │                                   │          │
│ 侧   │                                   │  AI      │
│ 边   │         主内容区                    │  Agent   │
│ 栏   │      (router-view)                │  面板    │
│      │                                   │  320px   │
│ 56px │                                   │          │
│      │                                   │          │
└──────┴───────────────────────────────────┴──────────┘
```

**顶栏 (48px)：**
- 左: Logo (点击回 dashboard) + 当前项目名(可切换) 或 面包屑
- 右: 积分余额(带充值按钮) + 通知铃铛 + 用户头像(下拉菜单)

**侧边栏 (56px 收起 / 200px 展开)：**
- 图标导航，hover 显示文字标签
- 项目列表 / 当前项目(子菜单: 设定/画布/记录) / 设置
- 底部: 折叠/展开按钮

**AI Agent 面板 (320px)：**
- 右侧常驻，可通过按钮折叠
- 折叠时显示一个浮动按钮唤出

### 5.4 全局设定 (/project/:id/setup)

**这是核心页面，参考 nelwen.cn 的分步引导设计。**

**整体布局：**
```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  ①基本信息  ──  ②画面比例  ──  ③风格预设  ──  ④导演指导  │   │
│  │  (步骤条，当前步骤高亮，已完成步骤打勾)              │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │                                                 │   │
│  │              当前步骤内容区                       │   │
│  │              (居中，最大宽度 720px)               │   │
│  │                                                 │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│              [上一步]          [下一步 →]                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### Step 1: 基本信息
- 项目名称 (必填，输入框)
- 项目描述 (选填，textarea，2-3行)
- 简洁，两个字段即可

#### Step 2: 画面比例
- 三个视觉卡片横排
- 每个卡片内：比例示意图形 + 文字标签 + 推荐场景
```
┌─────────┐  ┌─────────┐  ┌─────────┐
│  ┌───┐  │  │  ┌──┐   │  │  ┌─┐   │
│  │   │  │  │  │  │   │  │  │ │   │
│  │16:9│  │  │  │1:1│  │  │  │9│   │
│  │   │  │  │  │  │   │  │  │:│   │
│  └───┘  │  │  └──┘   │  │  │16│  │
│  横屏    │  │  方形    │  │  │ │   │
│ 视频/电影│  │ 社交媒体 │  │  └─┘   │
│         │  │         │  │  短视频  │
└─────────┘  └─────────┘  └─────────┘
```
- 选中状态：品牌色边框 + 背景微亮 + 右上角勾选

#### Step 3: 风格预设 ⭐ (最重要的步骤)
```
[全部] [动漫] [写实] [漫画] [3D] [自定义]  ← 分类 Tab

┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
│ 预览图  │ │ 预览图  │ │ 预览图  │ │ 预览图  │
│(正方形) │ │        │ │        │ │        │
│        │ │        │ │        │ │        │
├────────┤ ├────────┤ ├────────┤ ├────────┤
│赛博朋克 │ │吉卜力风 │ │美漫风格 │ │水墨国风 │
└────────┘ └────────┘ └────────┘ └────────┘

┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
│ 预览图  │ │ 预览图  │ │ 预览图  │ │  ＋    │
│        │ │        │ │        │ │        │
│        │ │        │ │ 自定义  │ │
├────────┤ ├────────┤ ├────────┤ │ 新建   │
│写实电影 │ │日系动画 │ │欧美卡通 │ │ 预设   │
└────────┘ └────────┘ └────────┘ └────────┘
```

**预设卡片交互：**
- 正常状态：圆角卡片，预览图 + 名称
- Hover：微放大(scale 1.02) + 阴影加深
- 选中：品牌色边框(2px) + 右上角 ✓ 角标 + 背景微亮
- 点击"新建预设"：弹出编辑弹窗（名称/提示词/上传参考图）

#### Step 4: 导演指导
- 全局导演指导词 (textarea，4-5行，带 placeholder 示例)
- 剧本规则 (textarea，3-4行，带 placeholder 示例)
- 底部提示："这些设定会影响 AI 生成的所有内容，可随时在设置中修改"

**设计要点：**
- 内容区居中，留足呼吸空间
- 步骤切换有左右滑动动效
- 可跳步（点击步骤条任意步骤）
- 最后一步按钮变为"完成创建"，点击后跳转画布

### 5.5 节点画布 (/project/:id/canvas)

**这是产品的核心工作区。**

**整体布局：**
```
┌──────────────────────────────────────────────────────┐
│ [+ 添加 ▾] [自动布局] [适应] [撤销] [重做]  │ [导出] │  工具栏 40px
├──────────────────────────────────────────────────────┤
│                                                      │
│                                                      │
│              React Flow 画布                          │
│              (无限画布，可缩放/平移)                    │
│                                                      │
│                                                      │
│                                                      │
│                                                      │
│  ┌────────┐                                          │
│  │Minimap │                                          │
│  └────────┘                                          │
└──────────────────────────────────────────────────────┘
```

**节点类型设计：**

#### 分镜节点 (Scene Node) — 最常用
```
┌─────────────────────────┐
│ #1 分镜                  │  ← 标题栏(可拖拽)
├─────────────────────────┤
│ ┌─────────────────────┐ │
│ │                     │ │  ← 缩略图区 (16:9比例)
│ │    生成的图片/视频    │ │     无内容时显示占位
│ │                     │ │
│ └─────────────────────┘ │
│ "角色A走进咖啡厅..."     │  ← 分镜描述(2行截断)
├─────────────────────────┤
│ ⏳ 等待生成  │ ▶ 生成    │  ← 状态栏 + 操作按钮
└─────────────────────────┘
    ●                   ●     ← 连接点(左入右出)
```

状态变化：
- 空白：灰色占位 + "点击生成"
- 生成中：进度条动画 + 百分比
- 完成：显示缩略图 + 绿色状态
- 失败：红色边框 + 错误提示 + 重试按钮

#### 角色节点 (Character Node)
```
┌───────────────┐
│   ┌─────┐    │
│   │ 头像 │    │  ← 圆形头像
│   └─────┘    │
│   角色名称    │
│  "主角·女·25" │  ← 简要描述
└───────────────┘
       ●             ← 输出连接点
```

#### 资产节点 (Asset Node)
```
┌───────────────┐
│ 🖼️ 背景图.png │
│ ┌───────────┐ │
│ │  缩略图    │ │
│ └───────────┘ │
│ 1920×1080     │
└───────────────┘
```

#### 输出节点 (Output Node)
```
┌─────────────────────┐
│ 📤 最终输出          │
├─────────────────────┤
│ 格式: MP4 1080p     │
│ 时长: 预计 2:30     │
│ 状态: 3/8 分镜完成   │
├─────────────────────┤
│ [合成视频]           │
└─────────────────────┘
```

**画布交互：**
- 空白处双击 → 快速添加分镜节点
- 拖拽连线 → 建立依赖关系
- 框选多个节点 → 批量操作
- 右键节点 → 上下文菜单(生成/编辑/复制/删除)
- 右键空白 → 添加节点菜单
- Ctrl+Z / Ctrl+Y → 撤销/重做
- 滚轮缩放，空格+拖拽平移

**节点编辑面板（点击节点后右侧滑出）：**

## 七、设计参考 & Mood Board

### 整体风格参考
1. **Runway** (runwayml.com) — AI 创作工具的暗色调 + 内容优先
2. **Linear** (linear.app) — 极简 SaaS，信息密度高但不拥挤
3. **Figma** — 面板布局、工具栏设计
4. **ComfyUI** — 节点画布的视觉风格
5. **Midjourney Web** — 生成记录的网格展示

### 关键设计原则
1. **内容优先** — 用户生成的图片/视频是主角，UI 退后
2. **信息密度** — 专业工具用户需要高效率，不要过多留白
3. **状态清晰** — 生成中/完成/失败 一眼可辨
4. **操作就近** — 操作按钮靠近内容，减少鼠标移动
5. **渐进披露** — 高级选项默认收起，需要时展开
